・モブ敵やアイテム、背景にリストを使うことで複数の出現に対応しやすく、一つ一つの動きを管理しやすくした。
・画面外に出たらリストから消したり、背景などをループさせて使うことで、画面に映っていない場所での無駄な処理を行わないようにした。
